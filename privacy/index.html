Privacy Policy

Baseline Collection Checker

Baseline Collection Checker is an app designed to help merchants manually identify products that are not assigned to any collection in their store.

Data Access

The app accesses the following store data through the Shopify Admin API:

  • Product information (read-only)
  • Collection assignments (read-only)
  • Store identification information required for authentication

The app does not access customer data, orders, or payment information.

Data Usage

Product and collection data is accessed only when a merchant manually initiates a scan inside the app. During a scan, the app checks whether products are assigned to any collections. If a product has no collection assignment, an event record may be created for tracking purposes.

No automatic scans, background monitoring, or scheduled jobs are performed.

Data Storage

The app stores minimal operational data required for functionality, including:

  • Store identifier
  • Missing-collection event records (product ID, SKU, vendor)
  • Event timestamps (first detected, last checked, resolved date if assigned later)
  • Scan rule settings (if configured by the merchant)

This data is used solely to support event history, filtering, and CSV export features. The app does not store customer personal data.

When the app is uninstalled, associated operational data for the store is deleted.

Data Modification

Baseline Collection Checker is a read-only app. It does not modify:

  • Products
  • Collections
  • Inventory
  • Orders
  • Store settings

Data Sharing

The app does not sell, share, transfer, or disclose store data to third parties.

Security

The app uses Shopify session token authentication and follows Shopify’s app security guidelines. No background data processing or external data sharing occurs.

Contact

For questions regarding this privacy policy, please contact the app developer through the Shopify App Store listing.
